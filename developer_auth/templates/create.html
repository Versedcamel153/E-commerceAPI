POST /api/products/categories/all/
Requires Auth

Creates a new product category with the provided details.

Authentication

API-KEY: <YOUR_API_KEY>

Request Body

{
    "name": "string (required)",
    "description": "string (required)"
}

Responses

201 Created 
{
    "id": 1,
    "name": "pencil",
    "description": "Writing"
}
400 Bad Request 
{
    "error": "InvalidData",
    "message": "Name and description are required fields."
}

Example Request

curl -X POST https://E-commerceAPI.com/api/products/categories/all/ \
    -H "API-KEY: YOUR_API_KEY" \
    -H "Content-Type: application/json" \
    -d '{"name": "pencil", "description": "Writing"}'

Python Example

import requests

response = requests.post(
    "https://E-commerceAPI.com/api/products/categories/all/",
    headers={"API-KEY": "YOUR_API_KEY"},
    json={"name": "pencil", "description": "Writing"}
)
