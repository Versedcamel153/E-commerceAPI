Update Review

Update an existing review for a product.

PUT /api/products/<slug:slug>/reviews/<int:pk>/update/
Auth Required

Authentication

Header: API-KEY: <your_api_key>

Path Parameters

Parameter Type Description
slug string The unique slug of the product.
pk integer The unique identifier of the review.

Request Body

Field Type Description
rating integer Rating value (1-5).
comment string Review comment text.

Responses

200 OK 
{
    "id": 1,
    "created_at": "17 January 2025",
    "user_id": 1,
    "username": "johndoe",
    "rating": 5,
    "comment": "Updated review comment.",
    "product": 1
}
400 Bad Request 
{
    "error": "ValidationError",
    "message": "Rating must be between 1 and 5."
}
404 Not Found 
{
    "error": "ReviewNotFound",
    "message": "The requested review does not exist."
}
cURL 
curl -X PUT "https://api.ecommerceapi.com/api/products/pencil/reviews/1/update/" \
  -H "API-KEY: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "rating": 5,
    "comment": "Updated review comment."
  }'
Python (requests) 
import requests

url = "https://api.ecommerceapi.com/api/products/pencil/reviews/1/update/"
headers = {
    "API-KEY": "your_api_key",
    "Content-Type": "application/json"
}
data = {
    "rating": 5,
    "comment": "Updated review comment."
}

response = requests.put(url, headers=headers, json=data)
print(response.json())
