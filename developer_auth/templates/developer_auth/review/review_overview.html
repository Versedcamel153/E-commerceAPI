Review Endpoint

The review endpoint allows you to manage product reviews. You can create, retrieve, update, and delete reviews.

Response Schema: application/json

Review Object

Field Type Description
id Integer A unique identifier for the review.
created_at String (date-time) Timestamp when the review was created.
user_id Integer A unique identifier for a user.
username String A name identifier for a user. Defaults to "Anonymous".
rating Integer (1-5) The number of stars for a rating.
comment String The body for the review.
product Integer ID for the product associated with the review.
