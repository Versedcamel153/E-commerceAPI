Review Enpoint

Details:

This explains the everythin about the review endpoint.

Response Schema: application/json

id

Integer

A unique identifier for the review.

created_at

String(date-time)

Timestamp when the review was created.

user_id

Integer(int64)

A unique identifier for a user.

username

String

A name identifier for a user. If not provided, the API uses "Anonymous" by default.

rating

Integer(1, 2, 3, 4, 5)

The number of stars for a rating. 1 - 1 Star, 2 - 2 Stars, 3 - 3 Stars, 4 - 4 Stars, 5 - 5 Stars

comment

String

The body for the review.

product

Integer

ID for the product associated with the review.
