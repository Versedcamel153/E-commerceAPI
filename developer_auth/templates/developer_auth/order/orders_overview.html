Order Enpoint

Details:

This explains the everythin about the order endpoint.

Response Schema: application/json

id

Integer

A unique identifier for the order.

items

Array

An object of the items in the order.

item_count

String

Number of items/products in the order.

created_at

String(date-time)

Timestamp when the order was created.

user_id

Integer(int64)

The unique User to whom the order is for.

total_price

decimal(^-?\d{0,8}(?:\.\d{0,2})?$)

Total price of the items in the order.

status

String(Options: 'PENDING', 'CANCELLED', 'COMPLETED')

Status of the order.

cart

Integer

ID of the cart which contains the items for the order.

Request Schema:

cart

(Required)

Integer

ID of the cart which contains the items for the order.
