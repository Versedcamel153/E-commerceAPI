Orders Overview

Manage customer orders, including creation, retrieval, and cancellation.

Order Object

Represents a customer's order in the system.

Field Type Description
id integer Unique identifier for the order.
items array List of items included in the order.
item_count integer Total number of items in the order.
created_at string (date-time) Timestamp when the order was created.
user_id integer ID of the user who placed the order.
total_price decimal Total cost of all items in the order.
status string Current status: PENDING, CANCELLED, or COMPLETED.
cart integer ID of the source cart for this order.
