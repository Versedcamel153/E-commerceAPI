Update Category

Updates an existing product category.

PUT /api/products/categories/<slug:slug>/
Auth Required

Authentication

Header: API-KEY: <your_api_key>

Path Parameters

Parameter Type Description
slug string The unique identifier for the category.

Request Body

JSON 
{
    "name": "string (optional)",
    "description": "string (optional)"
}

Responses

200 OK 
{
    "id": 1,
    "name": "updated-pencil",
    "description": "Updated description"
}
400 Bad Request 
{
    "error": "ValidationError",
    "message": "At least one field (name or description) must be provided."
}
cURL 
curl -X PUT "https://api.ecommerceapi.com/api/products/categories/pencil/" \
  -H "API-KEY: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "updated-pencil",
    "description": "Updated description"
  }'
Python (requests) 
import requests

url = "https://api.ecommerceapi.com/api/products/categories/pencil/"
headers = {
    "API-KEY": "your_api_key",
    "Content-Type": "application/json"
}
data = {
    "name": "updated-pencil",
    "description": "Updated description"
}

response = requests.put(url, headers=headers, json=data)
print(response.json())
