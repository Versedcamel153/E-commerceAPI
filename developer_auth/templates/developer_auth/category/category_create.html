Create Category

Creates a new product category with the provided details.

POST /api/products/categories/all/
Auth Required

Authentication

Header: API-KEY: <your_api_key>

Request Body

JSON 
{
    "name": "string (required)",
    "description": "string (required)"
}

Responses

201 Created 
{
    "id": 1,
    "name": "pencil",
    "description": "Writing"
}
400 Bad Request 
{
    "error": "InvalidData",
    "message": "Name and description are required fields."
}
cURL 
curl -X POST "https://api.ecommerceapi.com/api/products/categories/all/" \
  -H "API-KEY: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "pencil",
    "description": "Writing"
  }'
Python (requests) 
import requests

url = "https://api.ecommerceapi.com/api/products/categories/all/"
headers = {
    "API-KEY": "your_api_key",
    "Content-Type": "application/json"
}
data = {
    "name": "pencil",
    "description": "Writing"
}

response = requests.post(url, headers=headers, json=data)
print(response.json())
