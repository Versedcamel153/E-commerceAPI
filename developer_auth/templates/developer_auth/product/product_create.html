Create Product

Creates a new product with the provided details.

POST /api/products/create/
Auth Required

Authentication

Header: API-KEY: <your_api_key>

Request Body

JSON 
{
    "images": [],   
    "name": "pencil",   
    "description": "Writing",   
    "price": "5",   
    "stock_quantity": 100,  
    "category": "stationary"    
}

Responses

201 Created

    "reviews": [],
    "images": [],
    "category_id": 1,
    "category_name": "stationary",
    "average_rating": "No reviews yet",

    "message": "Name and description are required fields."
}
cURL 
curl -X POST "https://api.ecommerceapi.com/api/products/create/" \
  -H "API-KEY: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "pencil",
    "description": "Writing",
    "price": "5",
    "stock_quantity": 100,
    "category": "stationary"
  }'
Python (requests) 
import requests

url = "https://api.ecommerceapi.com/api/products/create/"
headers = {
    "API-KEY": "your_api_key",
    "Content-Type": "application/json"
}
data = {
    "name": "pencil",
    "description": "Writing",
    "price": "5",
    "stock_quantity": 100,
    "category": "stationary"
}

response = requests.post(url, headers=headers, json=data)
print(response.json())
