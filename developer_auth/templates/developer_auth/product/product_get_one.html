Get Single Product

Retrieves details of a specific product by its slug.

GET /api/products/<slug:slug>/
Auth Required

Authentication

Header: API-KEY: <your_api_key>

Path Parameters

Parameter Type Description
slug string The unique identifier for the product. Usually a URL-friendly version of the product's name.

Responses

200 OK 
{
    "id": 1,
    "reviews": [],
    "images": [],
    "category_id": 1,
    "category_name": "Electronics",
    "average_rating": "No reviews yet",
    "total_reviews": 0,
    "name": "Smartphone",
    "slug": "smartphone",
    "description": "A high-end smartphone.",
    "price": "799.00",
    "stock_quantity": 300,
    "created_date": "2023-10-27T10:00:00Z",
    "category": 1
}
404 Not Found 
{
    "error": "ProductNotFound",
    "message": "The requested product does not exist."
}
cURL 
curl -X GET "https://api.ecommerceapi.com/api/products/smartphone/" \
  -H "API-KEY: your_api_key"
Python (requests) 
import requests

url = "https://api.ecommerceapi.com/api/products/smartphone/"
headers = {
    "API-KEY": "your_api_key"
}

response = requests.get(url, headers=headers)
print(response.json())
