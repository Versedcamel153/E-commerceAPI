Product Endpoint

The product endpoint allows you to manage products in your e-commerce store. You can create, retrieve, update, and delete products.

Response Schema: application/json

Product Object

Field Type Description
id Integer A unique identifier for the product.
name String Name of the product.
description String Detailed description of the product.
slug String URL-friendly identifier for the product.
price Decimal Price of the product.
stock_quantity Integer Quantity of the product available in stock.
category Integer ID of the category the product belongs to.
category_name String Name of the category the product belongs to.
images Array List of images associated with the product.
reviews Array List of reviews associated with the product.
average_rating String Average customer rating for the product.
total_reviews String Total number of reviews for the product.
created_date String (ISO 8601) Timestamp when the product was created.

Request Schema

Field Type Description
name * String Name of the product.
description * String Detailed description of the product.
price * Decimal Price of the product.
stock_quantity * Integer Quantity of the product available in stock.
category * Integer ID of the category the product belongs to.
slug String URL-friendly identifier. Auto-generated if not provided.
images Array List of images associated with the product.
