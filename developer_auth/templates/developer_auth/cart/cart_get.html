Get Cart

Retrieve the current user's shopping cart.

GET /api/cart/
Auth Required

Authentication

Header: API-KEY: <your_api_key>

Responses

200 OK 
{
    "id": 1,
    "items": [],
    "total_quantity": 0,
    "total_price": 0,
    "user_id": 1,
    "created_at": "2025-01-17T10:42:15.082919Z"
}
404 Not Found 
{
    "error": "CartNotFound",
    "message": "The requested cart does not exist."
}
cURL 
curl -X GET "https://api.ecommerceapi.com/api/cart/" \
  -H "API-KEY: your_api_key"
Python (requests) 
import requests

url = "https://api.ecommerceapi.com/api/cart/"
headers = {
    "API-KEY": "your_api_key"
}

response = requests.get(url, headers=headers)
print(response.json())
