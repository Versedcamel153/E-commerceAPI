Cart Overview

Manage shopping carts, add items, and process orders.

Cart Object

Field Type Description
id integer A unique identifier for the cart.
items array List of items/products in the cart.
total_quantity integer Total number of items in the cart.
total_price string Total price of all items in the cart.
user_id integer ID of the user who owns the cart.
created_at string (date-time) Timestamp when the cart was created.

Add Item Schema

Field Type Description
product integer ID of the product to add to the cart. REQUIRED
quantity integer Quantity of the product to add. REQUIRED
