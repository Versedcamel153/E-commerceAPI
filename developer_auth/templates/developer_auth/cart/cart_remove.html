Remove Cart Item

Remove an item from the shopping cart.

DELETE /api/cart/<int:pk>/remove/
Auth Required

Authentication

Header: API-KEY: <your_api_key>

Path Parameters

Parameter Type Description
pk integer The unique identifier of the cart item to remove.

Responses

204 No Content

The item was successfully removed from the cart. No content is returned.

404 Not Found 
{
    "error": "ItemNotFound",
    "message": "The requested cart item does not exist."
}
cURL 
curl -X DELETE "https://api.ecommerceapi.com/api/cart/1/remove/" \
  -H "API-KEY: your_api_key"
Python (requests) 
import requests

url = "https://api.ecommerceapi.com/api/cart/1/remove/"
headers = {
    "API-KEY": "your_api_key"
}

response = requests.delete(url, headers=headers)
print(response.status_code)
