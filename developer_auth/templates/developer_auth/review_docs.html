Create a review

URL:

https://E-commerceAPI.com/api/products/<slug:slug>/reviews/create/

Methods:

POST

Authentication:

Headers
API-Key: {your-APIKEY}

Request Body: 

                
{
"user_id": 1,
"username": "...",
"rating": 1,
"comment": "...",
"product": 1
}

Response:

Status Code:201 Created

Response Body
                
{
"user_id": 1,
"username": "...",
"rating": 4,
"comment": "...",
"product": 1
}

Delete a review

URL:

https://E-commerceAPI.com/api/products/<slug:slug>/reviews/delete/

Methods:

DELETE

URL Parameters:

Slug (String, Required): The unique identifier for the category. This is usually a URL-friendly version of the category's name (e.g., "Electronics"). It is automatically generated upon creation of a category.

Authentication:

Headers
API-Key: {your-APIKEY}

Response:

Status Code":204 No Content
