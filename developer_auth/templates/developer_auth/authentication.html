Authentication

Our API uses API Key authentication to secure access. Each developer must generate an API key after signing up and use it to authenticate requests. The API key should be passed as a header API-Key in every request.

Ensure your API key is kept confidential and not exposed in public code repositories.

An API Key is needed to access all the endpoints. You can obtain one from the API Keys section of your dashboard.

Example Usage

Python

python 
import requests        

url = "https://api.ecom-api.com/v1/products"
headers = {
    "API-KEY": "YOUR_API_KEY",
}

response = requests.get(url, headers=headers)

cURL

bash 
curl -X GET "https://api.ecom-api.com/v1/products" \
  -H "API-KEY: YOUR_API_KEY" \
  -H "Content-Type: application/json"
